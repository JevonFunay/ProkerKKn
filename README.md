# E-Portofolio KKN Kelompok 44 — Dusun Karangnongko

Website ini merupakan pusat dokumentasi digital untuk merekam seluruh rangkaian kegiatan dan pengabdian masyarakat Kuliah Kerja Nyata (KKN) Kelompok 44 dari Universitas Kristen Duta Wacana di Dusun Karangnongko, Pacitan.

## Tujuan Website

1. **Arsip Digital Terpadu:** Menjadi rekam jejak digital yang merangkum implementasi program kerja strategis, termasuk penguatan mitigasi bencana alam dan digitalisasi wilayah pesisir.
2. **Promosi Potensi Daerah:** Mengangkat potensi lokal Dusun Karangnongko, mulai dari pariwisata alam hingga produk unggulan UMKM binaan agar lebih dikenal luas.
3. **Referensi Keberlanjutan:** Menjadi basis data dan panduan yang dapat dimanfaatkan oleh perangkat desa atau angkatan KKN selanjutnya untuk menyusun program berkelanjutan.

*Dibuat oleh Kelompok 44 KKN UKDW (Periode Juni - Juli 2026).*
